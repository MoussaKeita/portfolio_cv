Project 1: Chatbot Assistant

Un chatbot, appelé parfois assistant virtuel, est un programme informatique qui simule une conversation (« chat » en anglais) avec une personne, à l’écrit ou à l’oral. Pour ce projet j’ai construis un assistant virtuel capable de repondre par Ecrit une personne lorsqu’on lui pose une question dont il a connaissance. Pour ce projet j’ai utilisé un fichier json qui va jouer le rôle d’une base de données, ensuite grâce à des techniques de NLP(Natural Langage Processsing) j’ai tokenizé les données avant de les passées à mon réseau de neurone Deep Learning pour enfin faire une prediction .
